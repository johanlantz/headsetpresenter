How to activate logging in HeadsetPresenter

 

In order to understand your specific problem you have been asked by support to provide a log file from the HeadsetPresenter. This is a very simple procedure.

 

  1. Start the application and click on the Settings button in the main dialog.
  2. In the settings dialog. Hold the CTRL key and click left mouse button anywhere on the screen.
    This message will show:
  3. Restart HeadsetPresenter.
  4. A new program called SS_Log will start and it will automatically generate a log file when you use HeadsetPresenter. You do not need to do anything with the SS_Log application.
  5. Now use the HeadsetPresenter as far as you can up until your problem happens.
  6. Save the file in SS_log and send it to support@headsetpresenter.com
